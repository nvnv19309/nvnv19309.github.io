# suwuyuan 个人网站

这是一个从 0 开始搭建的静态个人网站，第一版只使用 HTML、CSS 和少量 JavaScript。

## 本地查看

直接双击 `index.html`，或在浏览器中打开这个文件即可预览。

## 怎么改内容

- 改文字：打开 `index.html`，搜索对应分区标题，例如“个人简介”“个人作品”。
- 改样式：打开 `styles.css`，优先修改顶部 `:root` 里的颜色变量。
- 改交互：打开 `script.js`，目前只包含移动端导航和作品分类筛选。
- 换头像：把图片放进 `assets/`，再替换 `index.html` 里的 `assets/portrait-placeholder.svg`。
